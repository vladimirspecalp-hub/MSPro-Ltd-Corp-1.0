# Статистики

Статистика — числовой показатель производства ЦКП за период.
Это единственный объективный инструмент управления по Хаббарду.

## Правила формулировки

1. Считает только ЗАВЕРШЁННЫЕ продукты (не «в процессе»)
2. Один пост = одна главная статистика (можно вспомогательные)
3. Период = неделя (четверг 14:00 → четверг 14:00)
4. Должна расти при росте производства (не инверсная)

## График

- Вертикальная ось: значение статистики
- Горизонтальная: недели
- Рисуется КАЖДУЮ неделю без пропусков
- Тренд определяется визуально по последним 3-4 точкам

## Определение тренда

| Тренд | Что значит |
|-------|-----------|
| Резко вверх (+30% и более) | Изобилие или Власть |
| Плавно вверх (+5..+30%) | Норма |
| Плоско (±5%) | Зависит: если высоко — Норма, если низко — ЧП |
| Плавно вниз (−5..−30%) | Чрезвычайное Положение |
| Резко вниз (более −30%) | Опасность |
| Ноль или около нуля | Не-существование |

## ИЦО — Индекс Целевой Отдачи

ИЦО = (Факт / План) × 100%. Показывает эффективность относительно цели.
- ИЦО > 120% — пересмотреть план вверх
- ИЦО 80-120% — в норме
- ИЦО < 80% — применить формулу состояния

## Ошибки

- «Количество звонков» — это действие, не продукт. Лучше: «Назначенные встречи»
- Менять формулировку статистики чаще раза в квартал — теряется история
- Не вести график — управление становится «по ощущениям»
