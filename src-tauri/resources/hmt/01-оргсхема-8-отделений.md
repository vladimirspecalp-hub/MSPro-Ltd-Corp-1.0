# Оргсхема — 8 отделений

Организующая схема компании по Хаббарду состоит из 8 отделений (0–7).
Каждое отделение содержит 3 отдела. Поток производства идёт слева направо:
от получения заказа до доставки и денег.

## Таблица отделений

| № | Название | Функция |
|---|----------|---------|
| 0 | Офис Владельца | Стратегия, инспекции, координация всей компании |
| 1 | HCO (Организационное) | Найм, посты, коммуникации, этика |
| 2 | Распространение | Маркетинг, продажи, управление доходом |
| 3 | Финансовое | Бухгалтерия, казначейство, учёт активов |
| 4 | Техническое (Производство) | Основная деятельность — выполнение заказов |
| 5 | Квалификация | Обучение, контроль качества, исправление |
| 6 | Связи с общественностью | PR, репутация, работа с территорией |
| 7 | Исполнительное | Высшее руководство, планирование, координация руководителей |

## Поток производства

1. **Отд. 2** (Распространение) — находит клиента, заключает договор
2. **Отд. 3** (Финансы) — принимает оплату, выделяет бюджет
3. **Отд. 4** (Техническое) — выполняет работу, создаёт продукт
4. **Отд. 5** (Квалификация) — проверяет качество
5. **Отд. 6** (PR) — публикует результат, работает с обратной связью

Отделения 0, 1, 7 — обеспечивающие (управление, кадры, координация).

## 3 отдела в каждом отделении

Каждое отделение делится на 3 отдела (секции). Например:
- Отд. 1 HCO: Найм → Коммуникации → Инспекции/Этика
- Отд. 4 Техническое: Планирование → Производство → Контроль

## Принцип

Оргсхема — это не «иерархия начальников», а **маршрут частицы** (заказа/проекта)
через компанию. Если частица застревает — ищи пустой пост на этом участке маршрута.
