# Финансовое планирование (СФП)

СФП — Система Финансового Планирования по Хаббарду. Основной принцип:
деньги распределяются ПО ФОРМУЛЕ, а не «кому громче надо».

## 6 счетов (фондов)

| # | Счёт | Назначение | Приоритет |
|---|------|-----------|-----------|
| 1 | Счёт дохода | Вся входящая выручка | Сбор |
| 2 | Счёт расходов | Операционные расходы (ЗП, аренда, связь) | 1-й |
| 3 | Резервный фонд | Неприкасаемый запас (3 месяца расходов) | 2-й |
| 4 | Фонд оборудования | Инвестиции в средства производства | 3-й |
| 5 | Фонд развития | Маркетинг, обучение, новые проекты | 4-й |
| 6 | Фонд владельца | Дивиденды, личный доход владельца | 5-й |

## Правило распределения

1. Весь доход поступает на Счёт дохода
2. Сначала закрой Счёт расходов (обязательства)
3. Остаток распределяй по приоритету: резерв → оборудование → развитие → владелец
4. Фонд владельца = ПОСЛЕДНИЙ (не первый!)

## Точка безубыточности

Минимальная выручка при которой покрываются все расходы (Счёт 2).
Рассчитать: сумма всех постоянных расходов + средние переменные.
Если доход ниже точки безубыточности 2 недели подряд — состояние ЧП для Отд. 3.

## Правила

- Никогда не тратить из Резервного фонда без письменного решения Владельца
- Счёт расходов = фиксированный бюджет, утверждённый на квартал
- При Изобилии — увеличивать Резервный фонд, НЕ расходы
- При ЧП — первым урезается Фонд развития, последним — расходы на ЗП

## Еженедельный ритуал

Каждый четверг (после закрытия статистик):
1. Зафиксировать доход за неделю
2. Распределить по фондам
3. Утвердить план расходов на следующую неделю
4. Сверить баланс резервного фонда с нормативом
