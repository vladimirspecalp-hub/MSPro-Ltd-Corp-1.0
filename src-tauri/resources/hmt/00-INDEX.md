# Справочник HMT — Указатель тем

Система управления MSPro-Ltd Corp основана на технологии Хаббарда в интерпретации Высоцкого.

## Темы

| # | Тема | Описание |
|---|------|----------|
| 01 | оргсхема-8-отделений | Структура из 8 отделений, поток производства |
| 02 | цкп | Ценный Конечный Продукт — 4 критерия |
| 03 | обязанности-владельца | 7 функций владельца компании |
| 04 | обязанности-руководителя | 2 фазы, 10 обязанностей |
| 05 | статистики | Формулировка, график, тренд, ИЦО |
| 06 | формулы-состояний | 8 состояний с пошаговыми формулами |
| 07 | координация | Совещания, повестка, рудименты |
| 08 | стратегическое-планирование | 6 этапов, иерархия документов |
| 09 | финансовое-планирование | СФП, 6 счетов, фонды |

## Использование

CEO вызывает `read_hmt_topic(topic)` для получения детального справочника по теме.
Файлы редактируемы Владельцем через проводник: `<AppData>/Vault/01-HMT-Knowledge/`.
