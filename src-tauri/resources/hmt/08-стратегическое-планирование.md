# Стратегическое планирование

Стратегический план — документ Владельца, определяющий куда компания движется
на 2-5 лет. Без него ежедневная работа не имеет направления.

## 6 этапов создания стратплана

1. **Замысел** — чего хочет Владелец (выручка, доля рынка, география, продукты)
2. **Анализ текущего** — где мы сейчас (финансы, люди, репутация, конкуренты)
3. **Разрыв** — что мешает перейти из текущего в замысел
4. **Стратегия** — как преодолеть разрыв (3-5 ключевых решений)
5. **Тактический план** — конкретные проекты по кварталам
6. **Программы** — детальные задачи со сроками и ответственными

## Иерархия документов

| Уровень | Документ | Горизонт | Кто |
|---------|----------|----------|-----|
| 1 | Замысел/Миссия | 5+ лет | Владелец |
| 2 | Стратегический план | 2-3 года | Владелец |
| 3 | Тактический план | Квартал | Директор |
| 4 | Боевой план | Неделя | Руководители |
| 5 | Задача | День | Исполнитель |

## Правила

- Стратплан пересматривается раз в год (не чаще)
- Тактический план — раз в квартал
- Боевой план — каждый понедельник на координации
- Каждый нижний уровень ДОЛЖЕН вытекать из верхнего

## Типичные ошибки

- Нет замысла — работа «по инерции» без направления
- Слишком много целей — распыление ресурсов (максимум 3-5 приоритетов)
- План без ответственного — документ для полки
- Нет контрольных точек — как понять что двигаемся?
